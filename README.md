#### 《计算机网络》下半学期课程笔记

#### 《计算机网络》期末考试考点（不带答案）
- 所有知识点都基于张进老师的课件，亲测覆盖了75%到85%的2019年秋季南科大计算机网络期末考试真题考点。并且加了两三道2019年秋季南科大计算机网络期末考试的考后回忆的真题（真题未显著标出）。仅含题目，供自测。
